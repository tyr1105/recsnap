## ✨ 这是什么？

RecSnap 是一个**完全免费、无需注册**的在线屏幕录制工具，可以直接在浏览器中录制屏幕并生成 GIF 或 WebM 动图。所有处理都在本地完成，**你的录屏数据永远不会上传到任何服务器**。

## 🎯 功能特点

- 🖥️ **屏幕录制** — 支持录制整个屏幕、窗口或浏览器标签页
- 🎞️ **多格式导出** — 支持 GIF 和 WebM 格式输出
- ⚡ **即时处理** — 浏览器端实时转码，无需等待服务器
- 🔒 **隐私安全** — 100% 本地处理，数据不出设备
- 🚀 **无需注册** — 打开即用，没有任何限制
- 💰 **完全免费** — 无隐藏收费，无水印

## 🛠️ 使用方法

1. 打开 [RecSnap](https://tyr1105.github.io/recsnap/)
2. 选择要录制的屏幕/窗口/标签页
3. 点击录制按钮开始录屏
4. 停止录制后自动生成 GIF/WebM
5. 一键下载或复制到剪贴板
